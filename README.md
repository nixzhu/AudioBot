# AudioBot

AudioBot helps you do audio record & playback.

## Requirements

Swift 2.2, iOS 8.0

## Usage

See the demo.

## Installation

### CocoaPods

```ruby
pod 'AudioBot', '~> 0.4.0'
```

### Carthage

```ogdl
github "nixzhu/AudioBot" >= 0.4.0
```
