# AudioBot

AudioBot helps you do audio record & playback.

## Requirements

Swift 3, iOS 8

(Swift 2.3, use version 0.5.0)

## Usage

See the demo.

## Installation

### Carthage

```ogdl
github "nixzhu/AudioBot" >= 1.0.1
```

### CocoaPods

```ruby
pod 'AudioBot', '~> 1.0.1'
```
